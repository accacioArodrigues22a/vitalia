Como jogar?

Objetivo

Colete todas as frutas espalhadas pelo mapa sem perder suas 3 vidas!

Dicas

Fuja dos monstros: eles seguem você!

Aperte P no teclado para pausar o jogo.

Controle

    • Seta para Cima ou tecla W: mover para cima
    • Seta para Baixo ou tecla S: mover para baixo
    • Seta para Direita ou tecla D: mover para a direita
    • Seta para Esquerda ou tecla A: mover para a esquerda

Regras

- Desvie dos obstáculos para não ser pego!

- Fuja dos monstros! Se perder 3 vidas será Game Over!

- Colete todas as frutas para vencer a fase e avançar no jogo!

- Cada fase traz novos desafios: obstáculos diferentes, monstros mais rápidos e mais frutas!
